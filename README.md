# Pokedex
Premier Pokedex
## Avant tout
Lancer le script Bash pre_install.sh avec la commande ./pre_install.sh

## Lancer le projet 
Utiliser les commandes 
* export FLASK_APP=app 
* export FLASK_ENV=development

avant de vouloir lancer le projet avec "python3 -m flask run"
